# SPEC.md - La Michoacana Natural Fruit Landing Page

## 1. Concept & Vision

Landing page para Heladería La Michoacana Natural Fruit en California, USA. Diseñada para mostrar productos artesanales mexicanos (helados, paletas, frutas y antojos) con precios, además de atraer potenciales franquiciados. El sitio transmite frescura, autenticidad mexicana y oportunidades de negocio.

## 2. Design Language

**Aesthetic:** Vibrante, fresco y mexicano - colores vivos inspirados en frutas naturales y arte popular mexicano.

**Color Palette:**
- Primary: `#E91E63` (rosa mexicano / fresa)
- Secondary: `#FF9800` (naranja mango)
- Accent: `#4CAF50` (verde lima)
- Background: `#FFF8E1` (crema suave)
- Text: `#2D2D2D` (negro suave)
- Light: `#FFFFFF`

**Typography:**
- Headings: 'Fredoka One', cursive (Google Fonts) - divertida y mexicana
- Body: 'Nunito', sans-serif - legible y amigable

**Motion:** Animaciones suaves en scroll, hover en cards con elevación y scale.

## 3. Layout & Structure

1. **Header** - Logo, navegación sticky
2. **Hero** - Imagen de fondo con productos, título y CTA
3. **Productos** - Grid de productos con precios (Helados, Paletas, Frutas, Antojos, Bebidas)
4. **Franquicia** - Beneficios, inversión y ganancias potenciales
5. **Galería** - Fotos de productos (placeholder)
6. **Contacto** - Formulario para dudas de franquicias
7. **Footer** - Info de contacto y redes

## 4. Features & Interactions

- Navegación suave (smooth scroll)
- Cards de productos con hover effect
- Formulario de contacto funcional (Formspree)
- Diseño responsive (mobile-first)
- Animaciones de entrada en scroll

## 5. Component Inventory

**Header:**
- Logo a la izquierda
- Nav links: Inicio, Productos, Franquicia, Contacto
- Sticky con fondo sólido al scroll

**Hero:**
- Imagen de fondo (productos)
- Título con texto en español
- Botón CTA "Ver Productos"

**Product Card:**
- Imagen del producto
- Nombre
- Descripción breve
- Precio
- Hover: elevación y borde coloreado

**Franchise Section:**
- Beneficios (iconos)
- Información de inversión
- Potencial de ganancias
- CTA de contacto

**Contact Form:**
- Nombre completo
- Email
- Teléfono
- Mensaje
- Validación en tiempo real
- Éxito al enviar

## 6. Technical Approach

- HTML5 + CSS3 + Vanilla JS
- Formspree para formulario
- Google Fonts CDN
- Iconos: Lucide Icons
- Responsive: CSS Grid + Flexbox
- Mobile-first approach
